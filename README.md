# Análisis de pruebas A/B

## Descripción del proyecto

Este proyecto evalúa el desempeño de dos variantes de producto mediante un experimento A/B.
Se analizaron métricas de conversión y se aplicaron pruebas de hipótesis para determinar si las diferencias observadas eran estadísticamente significativas.

**Objetivo**

Determinar si una nueva variante de producto mejora la conversión de usuarios.

**Herramientas utilizadas**

 - Python
 - Pandas
 - SciPy
 - Análisis estadístico
- Jupyter Notebook

**Proceso del análisis**

1. Limpieza y validación de datos.
2. Análisis exploratorio de métricas.
3. Cálculo de tasas de conversión.
4. Aplicación de pruebas de hipótesis.
4. Interpretación de resultados.

**Resultados principales**

- Evaluación estadística de la diferencia entre variantes.
- Recomendación basada en datos sobre la mejor opción.

**Habilidades demostradas**

- Análisis estadístico
- Pruebas de hipótesis
- Interpretación de resultados
- Toma de decisiones basada en datos
